
# DND Search

This is a search front-end for the elasticsearch engine created by DND.
## Authors

- [Steven Scharf](https://www.linkedin.com/in/sscharf/)


## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
